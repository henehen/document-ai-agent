# 🤖 Document AI Agent

An AI-powered customer service agent that reads your business 
documents and answers customer questions instantly.

Built with Python, Groq, LangChain and Gradio.

---

## ✨ Features

- 📄 Reads multiple documents at once (PDF, DOCX, TXT)
- 💬 Answers customer questions instantly
- 🌍 Works in terminal AND as a web interface
- ⚡ Fast responses powered by Groq AI
- 🔒 Runs on your own API key
- 🆓 Completely free to run

---

## 🚀 Installation

### 1. Clone the repository
git clone https://github.com/yourusername/document-ai-agent.git
cd document-ai-agent

### 2. Install dependencies
pip install -r requirements.txt

### 3. Add your API key
Create a .env file and add:
GROQ_API_KEY=your_groq_api_key_here

Get your free Groq API key at: https://console.groq.com

### 4. Run the agent

Terminal version:
python agent.py

Web interface:
python app.py

---

## 💡 How It Works

1. Upload your business documents (FAQ, policies, manuals)
2. The AI reads and memorizes all the content
3. Customers ask questions in plain English
4. The agent answers instantly based on YOUR documents
5. Unknown questions are politely redirected to support

---

## 🛠️ Built With

- Python 3.11
- Groq API (LLaMA 3.3 70B)
- LangChain
- HuggingFace Embeddings

- Gradio
